# 🛒 SmartCart (C++ CLI E-Commerce System)

A command-line based E-Commerce system built using **C++**, demonstrating core concepts of **Object-Oriented Programming (OOP)** and **Data Structures (DSA)**.

---

## 🚀 Features
- View available products
- Add products to cart
- View cart with total price
- Menu-driven CLI interface

---

## 🧠 Concepts Used

### 🔹 Object-Oriented Programming
- Classes & Objects
- Encapsulation
- Basic Abstraction

### 🔹 Data Structures
- Dynamic storage using vectors (can be extended to Linked List)
- Searching operations

---

## 🛠️ Tech Stack
- Language: C++
- Interface: Command Line (CLI)
